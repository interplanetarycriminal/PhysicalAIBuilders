# Strange & archetypal

Ten personas who delight in the weird. Build one of these to convince your friends you've been replaced by a fun ghost.

---

## 81. The Cryptid Hunter — CSI ghost finder
**Asymmetry.** Finds unexplained presence via Wi-Fi CSI + mmWave overlap where no human could be.
**Build.** ESP32 (CSI) + LD2410B + mic. Find/Reason.
**Companion software.** A phone app that narrates the overlap. See [`../projects/ghost-hunter-csi-presence/`](../projects/ghost-hunter-csi-presence/).
**Why it's fun.** Scares friends at parties with *real sensor data*.

## 82. The Ghost Detective — RF séance
**Asymmetry.** Every "haunting" gets a plausible RF / magnetic / thermal explanation.
**Build.** ESP32 + magnetometer + AS3935 + thermal + EMF-sniffer on ADC. Find.
**Companion software.** Evidence board; LLM "debunker" voice persona that respects the client.
**Why it's fun.** Tour the weird with evidence.

## 83. The Cold-Case Sleuth — time-fold evidence correlator
**Asymmetry.** Correlate many sparse signals (BLE, temp, light) across weeks to find pattern breaks.
**Build.** ESP32 mesh across a house + retention DB. Find/Reason.
**Companion software.** See [`../projects/cold-case-sleuth-timefold/`](../projects/cold-case-sleuth-timefold/).
**Why it's fun.** Long-arc detective work that's satisfying at every checkpoint.

## 84. The Paranormal Skeptic — calibration toolkit
**Asymmetry.** Present hard calibration for "haunted" phenomena — CO₂, low-frequency sound, magnetic fields — for any room.
**Build.** ESP32-S3 + SCD41 + mic + magnetometer + IR thermometer. Find/Transform.
**Companion software.** Generates a printable "calibrated room report" branded to the skeptic's podcast.
**Why it's fun.** Content creator's dream prop.

## 85. The Improv Comedian — crowd-laugh meter
**Asymmetry.** Every joke's laughter power in real time — for post-show analysis.
**Build.** ESP32-S3 + mic + RMS + FFT + Wi-Fi. Transform/Distill.
**Companion software.** Set-review app that overlays set-list vs laughter; LLM coaches on material.
**Why it's fun.** Honest feedback, delivered kindly.

## 86. The Librarian — shelf beacon
**Asymmetry.** Each shelf e-paper beacon knows what books are missing.
**Build.** XIAO ESP32-C3 per shelf + light-gate array + e-paper. Find/Represent.
**Companion software.** Library OPAC integration; LLM generates a "books we're missing this week" display.
**Why it's fun.** Quiet competence, ambient magic.

## 87. The Pirate Radio Operator — solar FM relay
**Asymmetry.** Broadcast a low-power mesh-station with modest but determined reach.
**Build.** ESP32 + FM transmitter module + LoRa + solar + battery. Generate/Do.
**Companion software.** Queue manager; programme guide auto-generated.
**Why it's fun.** Old-soul project. Understand the responsibility (and local law) before you power up.

## 88. The Toy Maker — alive-in-your-hand plushies
**Asymmetry.** A soft toy that knows when it's being hugged vs shaken vs ignored.
**Build.** XIAO ESP32-C3 + IMU + capacitive touch + speaker. Find/Do/Represent.
**Companion software.** A small set of voices; LLM writes a daily "story from your friend".
**Why it's fun.** A gift that never gets put away.

## 89. The Cryptographer — offline secret-sharing
**Asymmetry.** Split a secret among friends using Shamir-scheme QR + BLE — no cloud needed.
**Build.** ESP32-S3 + screen + camera + BLE. Transform.
**Companion software.** Minimal open-source key-management viewer.
**Why it's fun.** The rare hobby that takes *more* data off the wire than it puts on.

## 90. The Conlanger — language diary sensor
**Asymmetry.** Records ambient speech phonetic stats (formants, rate, prosody) so the conlanger can spot natural-sounding features.
**Build.** XIAO ESP32-S3 Sense + mic + on-device VAD + Wi-Fi. Transform/Distill.
**Companion software.** Phoneme histogram + LLM that drafts "sounds your language should have".
**Why it's fun.** Craftwork on a timescale of decades.
