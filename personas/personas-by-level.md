# Personas, indexed by learning level

A shortcut for "I want something at my level." Level rubric: [`../framework/learning-levels.md`](../framework/learning-levels.md).

Each persona can be built at more than one level — we list the **entry-level** version here. Every build can be extended upward by adding more sensors, mesh, TinyML, LLM companions. Use the persona's page for the vibe; use this page for the ramp.

## How to read this

- **Flagship build** marker → there's a full guide in [`../projects/`](../projects/) for that exact rung.
- Personas without a flagship are a fine place to start contributing one. See [`../CONTRIBUTING.md`](../CONTRIBUTING.md) and the template in [`../projects/_template/`](../projects/_template/).
- Your *first* build should be at your current level. Your *second* build can jump +10 or +20; any further and you'll suffer.
